Execução migrations

Criação Migration
- dotnet ef migrations add Inicial2 --project . --startup-project ../FlowaStudy.WebApi

Atualização Migration no banco
- dotnet ef database update --project . --startup-project ../FlowaStudy.WebApi

Conectar Banco de Dados
- docker exec -it flowastudy_database psql -U developer -d flowastudy
